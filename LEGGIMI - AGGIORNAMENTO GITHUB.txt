AGGIORNAMENTO DEL SITO GIÀ PUBBLICATO

1. Aprire il repository GitHub “tp-room-2026” e restare nella sezione Code, branch main.
2. Cliccare Add file > Upload files.
3. Trascinare dentro la pagina TUTTO il contenuto di questa cartella, non la cartella esterna stessa.
4. GitHub mostrerà i file esistenti come sostituiti e il nuovo PDF nella cartella downloads.
5. Scrivere come messaggio: “Allinea Business Plan e investor room v2.1 — lancio AI rafforzato”.
6. Lasciare selezionato “Commit directly to the main branch” e cliccare Commit changes.
7. Attendere alcuni minuti e ricaricare il link già condiviso. Il link non cambia.
